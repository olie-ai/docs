## Filtros com o parâmetro `filter`

A API da Olie permite aplicar filtros dinâmicos nos endpoints que listam recursos, utilizando o parâmetro `filter`. Para utilizar, basta adicionar o parâmetro `filter` seguido do nome do campo que deseja filtrar.

<Tip>
Os filtros são uma maneira poderosa de refinar os resultados retornados pela API, permitindo que você obtenha exatamente os dados que precisa.
</Tip>

## Sintaxe básica

A sintaxe para aplicar filtros é simples e intuitiva:

```http
GET /api/v1/{endpoint}?filter[campo]=valor
```

### Exemplo prático

Para filtrar projetos pelo nome:

```http
GET /api/v1/projects?filter[name]=Nome do projeto
```

## Filtros múltiplos

Você pode combinar múltiplos filtros na mesma requisição para refinar ainda mais os resultados.

### Exemplo com múltiplos filtros

```http
GET /api/v1/projects?filter[name]=Projeto A&filter[status]=ativo
```

<Info>
Quando múltiplos filtros são aplicados, a API retorna apenas os recursos que atendem a **todos** os critérios especificados (operação AND).
</Info>

## Tipos de filtros

Os filtros funcionam com diferentes tipos de campos:

<AccordionGroup>
<Accordion title="Filtros de texto">
Para campos de texto, o filtro geralmente busca correspondências exatas:

```http
GET /api/v1/users?filter[email]=usuario@example.com
```
</Accordion>

<Accordion title="Filtros numéricos">
Para campos numéricos como IDs:

```http
GET /api/v1/tasks?filter[project_id]=123
```
</Accordion>

<Accordion title="Filtros de status">
Para campos de status ou enumerações:

```http
GET /api/v1/projects?filter[status]=concluido
```
</Accordion>

<Accordion title="Filtros de data">
Para campos de data, utilize o formato ISO 8601:

```http
GET /api/v1/projects?filter[created_at]=2024-01-15
```
</Accordion>
</AccordionGroup>

## Exemplo completo

Aqui está um exemplo completo de requisição com filtros:

<CodeGroup>
```bash cURL
curl -X GET "https://api.olie.ai/api/v1/projects?filter[status]=ativo&filter[name]=Meu Projeto" \
  -H "Authorization: Bearer SEU_TOKEN_AQUI"
```

```javascript JavaScript
const response = await fetch(
  'https://api.olie.ai/api/v1/projects?filter[status]=ativo&filter[name]=Meu Projeto',
  {
    headers: {
      'Authorization': 'Bearer SEU_TOKEN_AQUI'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://api.olie.ai/api/v1/projects"
params = {
    "filter[status]": "ativo",
    "filter[name]": "Meu Projeto"
}
headers = {
    "Authorization": "Bearer SEU_TOKEN_AQUI"
}

response = requests.get(url, params=params, headers=headers)
data = response.json()
print(data)
```
</CodeGroup>
