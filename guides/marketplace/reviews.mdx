## Como funcionam as avaliações

O sistema de avaliações permite que usuários compartilhem sua experiência com os templates adquiridos no Marketplace. As avaliações ajudam outros usuários a escolher os melhores templates e fornecem feedback valioso para os parceiros criadores.

## Quem pode avaliar

Qualquer usuário que **adquiriu** um template pode avaliá-lo. Não é necessário ter implantado o template para deixar uma avaliação.

## Formato da avaliação

Cada avaliação é composta por:

- **Estrelas** — Nota de 1 a 5 estrelas
- **Comentário** — Texto descrevendo a experiência com o template

As avaliações aparecem na página de detalhes do template no Marketplace, visíveis para todos os usuários.

## Moderação de avaliações

A moderação de avaliações é feita exclusivamente pelo time Olie.

<AccordionGroup>
<Accordion title="Ocultar avaliações">
  Podemos ocultar avaliações que violem as diretrizes da plataforma. Quando uma avaliação é ocultada:

  - As **estrelas** deixam de ser contabilizadas na média do template
  - O **texto** do comentário deixa de ser exibido
  - A avaliação não aparece mais na página do template
</Accordion>

<Accordion title="Parceiro não pode ocultar avaliações">
  O parceiro criador do template **não** tem a capacidade de ocultar avaliações diretamente. Se o parceiro considerar que uma avaliação é inadequada, deve solicitar uma revisão ao time Olie.
</Accordion>
</AccordionGroup>

<Warning>
Parceiros não podem ocultar avaliações dos seus templates. Se você encontrar uma avaliação que considere inadequada, entre em contato com o time Olie para solicitar uma revisão.
</Warning>

## Fluxo para parceiros

<Steps>
<Step title="Acompanhe as avaliações">
  Monitore as avaliações dos seus templates pelo painel do parceiro ou pela página do template no Marketplace.
</Step>

<Step title="Identifique feedbacks relevantes">
  Use os comentários para entender o que os usuários gostam e o que pode ser melhorado nos seus templates.
</Step>

<Step title="Solicite revisão se necessário">
  Caso identifique uma avaliação que viole as diretrizes da plataforma, entre em contato com o time Olie solicitando a revisão do comentário.
</Step>
</Steps>

<Tip>
Avaliações são uma ferramenta valiosa de feedback. Use-as para aprimorar seus templates e oferecer cada vez mais valor aos usuários do Marketplace.
</Tip>
