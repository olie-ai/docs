## Submetendo um template

Após [criar o template](/guides/marketplace/creating-template) com todos os metadados e selecionar o snapshot, você pode submetê-lo para revisão do time Olie. A submissão inicia o processo de moderação.

## Status do template

Após a submissão, seu template passará por um dos seguintes status:

<CardGroup cols={2}>
<Card title="Em análise" icon="clock">
  O template foi submetido e está aguardando revisão pelo time Olie. Nenhuma ação é necessária da sua parte.
</Card>

<Card title="Aprovado" icon="circle-check">
  O template foi aprovado e publicado no Marketplace. Usuários já podem visualizá-lo e adquiri-lo.
</Card>

<Card title="Alterações solicitadas" icon="pen">
  O moderador revisou o template e solicitou ajustes. Você receberá o feedback com as mudanças necessárias e poderá editar e resubmeter.
</Card>

<Card title="Rejeitado" icon="circle-xmark">
  O template foi rejeitado definitivamente pelo time Olie. Não é possível resubmeter um template rejeitado.
</Card>
</CardGroup>

<Warning>
A rejeição de um template é definitiva. Se o seu template for rejeitado, você precisará criar um novo template do zero caso deseje publicar o conteúdo no Marketplace.
</Warning>

## Notificações

Você será notificado por e-mail sempre que houver uma mudança de status no seu template (aprovação, solicitação de alterações ou rejeição).

## Ciclo de resubmissão

Quando o status do template muda para "Alterações solicitadas", você pode corrigi-lo e resubmeter. O ciclo funciona da seguinte forma:

<Steps>
<Step title="Receba o feedback">
  Leia o feedback do moderador com as alterações solicitadas. O feedback estará disponível no painel do parceiro.
</Step>

<Step title="Edite a empresa construtora">
  Faça os ajustes necessários na [empresa construtora](/guides/marketplace/template-builder).
</Step>

<Step title="Gere um novo snapshot">
  Após as alterações, [gere um novo snapshot](/guides/marketplace/snapshots) para capturar o estado atualizado do ambiente.
</Step>

<Step title="Atualize o template">
  Edite o template, selecione o novo snapshot e adicione uma justificativa explicando as alterações realizadas. Aproveite para editar as informações do template, como nome, descrição, categorias, imagens, etc.
</Step>

<Step title="Resubmeta para análise">
  Envie o template novamente para revisão. O status voltará para "Em análise".
</Step>
</Steps>

<Tip>
Inclua uma justificativa detalhada ao resubmeter, explicando como você endereçou cada ponto do feedback. Isso agiliza o processo de revisão.
</Tip>

Esse ciclo se repete até que o template seja aprovado ou rejeitado.

## Histórico de submissões

O painel do parceiro mantém um histórico completo das submissões, incluindo feedbacks dos moderadores e justificativas de resubmissão, permitindo acompanhar toda a evolução do template.
